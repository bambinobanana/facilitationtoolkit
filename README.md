# 💥 Movers Companion

A bilingual (EN/FR) facilitation tool for workshop facilitators, trainers, and youth programme leaders. Helps you find the right icebreaker or energizer for any session — in seconds.

Built for the [Movers Programme](https://www.undp.org) by UNDP Asia-Pacific.

---

## What it does

Facilitators often spend more time searching for the right activity than actually running it. Movers Companion solves that.

- **Smart Filter** — answer 5 quick questions (group size, familiarity, goal, time, space) and get matched activities instantly
- **Browse all** — explore the full library of 23 activities with filters by type and group size
- **Activity cards** — each activity includes step-by-step instructions, a facilitator script, a tip, and an expected outcome
- **My Favourites** — save your go-to activities with a heart tap, persisted across sessions
- **Copy & Export** — copy any activity to clipboard or save as PDF to share with co-facilitators
- **EN / FR** — full bilingual support, switch languages instantly

## Tech stack

- Plain HTML + React (via CDN) — no build step, no framework
- Babel standalone for JSX
- localStorage for favourites persistence
- Deployed on Vercel

---

## Deployment

This is a single-file app (`index.html`). To deploy:

1. Fork or clone this repo
2. Connect to [Netlify](https://netlify.com) via GitHub
3. Deploy with default settings — no build command needed

Every push to `main` triggers an automatic redeploy.
